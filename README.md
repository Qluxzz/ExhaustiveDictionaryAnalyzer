# Exhaustive Dictionary Analyzer in C#

Install [ExhaustiveDictionary.Analyzer](https://www.nuget.org/packages/ExhaustiveDictionary.Analyzer) [![NuGet](https://img.shields.io/nuget/v/ExhaustiveDictionary.Analyzer.svg)](https://www.nuget.org/packages/ExhaustiveDictionary.Analyzer) and place an `[Exhaustive]` attribute on an `Dictionary<TEnum, T>` to get exhaustive checks that you have added all members of the enum to the dictionary
